# Criar um card

> **Quem pode fazer**: Admin do quadro ou Editor.
> **Tempo estimado**: 1 minuto para o básico, mais alguns minutos se quiser detalhar.

## O que é um card

Card é a unidade de trabalho do KTask. Cada tarefa, demanda, pedido ou item a ser feito vira um card dentro de uma coluna de um quadro. O card carrega tudo o que importa sobre aquela tarefa: título, descrição, anexos, comentários, responsável, prazo, checklist, histórico de mudanças.

## Criar um card rápido

A maneira mais comum é direto na coluna:

1. Abra o quadro.
2. Vá até a coluna onde o card deve nascer (geralmente a primeira coluna, "A Fazer" ou "Briefing").
3. No rodapé da coluna, clique em **+ Adicionar card**.

   ![Botão + na coluna de entrada do quadro](/tutorial-para-clientes/img/15-criar-card.png)

4. Aparece um campo de texto na coluna. Digite o **título** do card (ex: "Criar arte para post de Páscoa").
5. Pressione **Enter** para salvar.

Pronto. O card é criado imediatamente e fica visível para todos no quadro. Você pode continuar adicionando outros cards na sequência sem fechar o campo.

> **Atalho útil**: pressione **Shift+Enter** dentro do campo de título para pular linha em vez de salvar (caso o título precise de duas linhas).

## Preencher detalhes do card

Depois de criar, clique no card para abrir o detalhe. Aí você pode preencher:

- **Descrição**: texto rico (negrito, lista, link, imagem inline) explicando o que precisa ser feito. Usa o editor Tiptap — atalhos comuns funcionam (Ctrl+B para negrito, Ctrl+I para itálico).
- **Líder**: a pessoa principal responsável pela tarefa. Aparece com avatar destacado no card.
- **Equipe**: outras pessoas envolvidas. Recebem notificações de mudanças no card.
- **Prazo**: data de entrega. Se passar, o card destaca em vermelho.
- **Etiquetas (tags)**: marcadores coloridos para categorizar (ex: "Urgente", "Cliente Alfa", "Refação").
- **Checklist**: lista de subtarefas dentro do card, com checkbox cada.
- **Anexos**: arquivos, imagens, links.
- **Status**: ativo, concluído, em espera ou cancelado (independente da coluna).

Tudo é salvo automaticamente conforme você digita — não tem botão "Salvar" explícito.

## Boas práticas no título

Um bom título resolve metade do trabalho de organização:

- **Específico**: "Post Instagram — promoção dia das mães — versão A" funciona melhor que "Post Instagram".
- **Curto**: até 80 caracteres. Se precisar contar mais, vai na descrição.
- **Sem emojis nem caracteres especiais**: dificulta busca depois.
- **Padrão pelo time**: combine com o time como nomear (ex: sempre começar com cliente, depois canal, depois conteúdo).

## Dúvidas comuns

**Posso criar um card em qualquer coluna?**
Sim, qualquer coluna do quadro aceita. Mas o costume é criar na primeira (entrada do fluxo) e deixar o card avançar pelas próximas conforme a tarefa progride.

**Esqueci de preencher algum campo. Perdi o card?**
Não. Só o título é obrigatório. Todo o resto pode ser preenchido depois, abrindo o card.

**O card aparece para todo mundo?**
Aparece para quem tem acesso ao quadro. Se você quer um card visível só para o time interno (sem o cliente), troque o status de privacidade do card para **Equipe** — assim só você (líder) e a equipe vinculada veem.

**Como sei se a pessoa que adicionei como líder foi avisada?**
Ela recebe notificação imediata (sino + WhatsApp se tiver telefone cadastrado e preferência ativa).

**Tem atalho de teclado para criar card?**
A criação é pelo campo inline na coluna — não há atalho global para "novo card" hoje.
